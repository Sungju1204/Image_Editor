# 이미지 보정 도구 (Image Correction Tool)

3D 프린터 이미지 데이터 보정

Vue.js 기반의 이미지 원근 보정(Perspective Correction) 웹 애플리케이션입니다.

## 주요 기능

- 📤 **다중 이미지 업로드**: 여러 이미지를 한 번에 업로드하고 순차적으로 처리
- ✏️ **선분 기반 보정**: 수평선 2개와 수직선 2개를 그려서 이미지의 원근을 보정
- 🎯 **직관적인 인터페이스**: 드래그 앤 드롭으로 선분을 쉽게 조정
- 📥 **결과 다운로드**: 보정된 이미지를 개별 또는 일괄 다운로드

## 기술 스택

- **Vue 3** - Composition API with `<script setup>`
- **Vite** - 빠른 개발 서버 및 빌드 도구
- **Canvas API** - 이미지 렌더링 및 원근 변환

## 사용 방법

1. **이미지 업로드**: 파일 선택 버튼을 클릭하거나 이미지를 드래그 앤 드롭
2. **선분 그리기**: 
   - 첫 번째 점을 클릭하여 선분 시작
   - 마우스를 움직여 방향 결정 (수평/수직 자동 감지)
   - 두 번째 점을 클릭하여 선분 완성
   - 수평선 2개, 수직선 2개를 그려주세요
3. **선분 조정**: 
   - 선분을 드래그하여 위치 조정
   - 선분 목록에서 "복사" 버튼으로 평행 선분 생성
   - "삭제" 버튼으로 선분 제거
4. **보정 완료**: 4개의 선분이 완성되면 "보정 완료" 버튼 클릭
5. **결과 확인 및 다운로드**: 보정된 이미지를 확인하고 다운로드

## 주요 기능 설명

### 원근 변환 (Perspective Transformation)
- Homography 행렬을 사용하여 4개의 교점을 기준으로 이미지를 평면으로 변환
- 순방향 및 역방향 매핑을 결합하여 최적의 결과 생성

### 선분 관리
- 수평/수직 선분 자동 감지
- 선분 드래그 앤 드롭으로 위치 조정
- 선분 복사 기능으로 평행선 쉽게 생성
- 실시간 교점 계산 및 표시
